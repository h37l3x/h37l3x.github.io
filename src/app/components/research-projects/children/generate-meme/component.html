Generate meme

Web app that allows you to create memes or work with images the way you want. For example, resize, crop, add stickers, place text with different styles etc.

Goal:

Create web app to easly create memes

Main functionality:
  • Creating collages with images, text blocks, stickers and other media available
  • Styling text (text size, color, font etc.)
  • Editing images (resize, crop, images combination)
  • Downloading resulting image
Code technologies I got involved with while working on this project:
  • HTML
  • CSS
  • Angular
